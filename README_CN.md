# Fair代币担保项目

[English](README.md) ・ [中文(简体)](README_CN.md)

## 介绍
该项目通过波场智能合约，为Fair代币持有者提供质押功能，通过质押为各种线上交易提供安全和信任。主要含以下功能：
 * 质押合约：Fair代币持有者发起质押合约，通过长期锁定代币获取信用，成为生态参与者，加入FAIR DAO 生态建设。
 * 争议合约：Fair代币消费者当遇到不公平对待时，可以提出争议，请求分割相应生态参与者质押的代币，通过Fair DAO公平投票，最终做出裁决。
 * 解押合约：生态参与者可以解押部分或全部代币，退出生态建设。
